# Dr. Mohammad Asaduzzaman Liton - ENT Specialist Website

This repository contains the source code for Dr. Mohammad Asaduzzaman Liton's professional website. He is an ENT (Ear, Nose, and Throat) Specialist and Head-Neck Surgeon working as Assistant Professor at Tangail Medical College.

## Project Overview

This is a responsive single-page website built with HTML, CSS, and JavaScript. It serves as a professional online presence for Dr. Liton, allowing patients to learn about his qualifications, experience, services offered, and book appointments.

## Features

- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Modern UI**: Clean, professional design with animations and transitions
- **Bengali Language Support**: Complete website in Bengali with proper font integration
- **Interactive Sections**:
  - Professional profile and biography
  - Education and work experience timelines
  - Medical services offered
  - Patient testimonials with slider
  - FAQ accordion section
  - Appointment booking form with validation
  - Contact information with map integration

## File Structure

- **index.html** - Main HTML document
- **styles.css** - Complete stylesheet with responsive design
- **script.js** - JavaScript functionality for interactive elements
- **images/** - Directory containing doctor's photos and logo
- **asaduzzaman.txt** - Text file with biographical information

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome for icons
- Google Fonts (Hind Siliguri for Bengali language support)
- Google Maps integration

## JavaScript Functionality

The website includes the following JavaScript functionality:
- Form validation for appointment booking
- Responsive navigation menu with smooth scrolling
- Testimonial slider with auto-rotation
- FAQ accordion toggle
- "Back to top" button
- Page loading animation
- Sticky navigation on scroll

## Getting Started

To run this project locally:

1. Clone the repository or download the files
2. Open the project in your code editor
3. Open index.html in your browser or use a local server
